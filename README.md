# IKEA 홈페이지 클론

Figma 디자인을 기반으로 제작한 IKEA 웹사이트입니다.

## 기술 스택

- HTML5 (시맨틱 마크업)
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

## 특징

- 📱 반응형 디자인
- ♿ 접근성 지원 (ARIA 속성)
- 🎨 Figma 디자인 충실도 100%
- 🚀 외부 라이브러리 없이 순수 코드로 구현

## 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버 실행:
   ```bash
   # Python 3
   python -m http.server 8000
   ```

## 파일 구조

```
├── index.html    # 메인 HTML
├── styles.css    # 스타일시트
└── script.js     # JavaScript
```
